# SEO On-Page Analysis

Evaluate the raw HTML and rendered content for standard on-page SEO best practices:
- **Title Tags**: Present, optimal length (50-60 characters), incorporates primary keywords.
- **Meta Descriptions**: Present, optimal length (150-160 characters), compelling CTA.
- **Headings**: Logical H1-H6 hierarchy. Exactly one H1 per page containing the core topic.
- **Image Optimization**: Images have descriptive `alt` text.
- **Internal Links**: Descriptive anchor text for internal navigation.

Generate a report formatted for `SEO-ONPAGE-REPORT.md`.

# SEO Keyword Strategy

Analyze the target domain's niche and content to produce a logical keyword strategy.
- Cluster keywords into Informational, Navigational, Commercial, and Transactional intents.
- Suggest Long-Tail opportunities with lower assumed difficulty.
- Identify primary and secondary keywords for the core landing pages.

Generate a report formatted for `SEO-KEYWORDS-REPORT.md`.

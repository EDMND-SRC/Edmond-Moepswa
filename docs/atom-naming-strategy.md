## Quick-Win Implementations

### Priority 1: Standard Naming Workflow (30 min per session)
```
1. Go to atom.com/business-name-generator
2. Input keywords (industry, core service, target market)
3. Select naming style (modern, professional, creative, etc.)
4. Generate 20–50 names
5. Filter for domain availability (.com preferred, .co/.io acceptable)
6. Shortlist 5–10 names based on:
   - Memorability (easy to say, spell, remember)
   - Relevance (hints at the business without being too literal)
   - Uniqueness (not similar to existing competitors)
   - Domain availability (.com available or reasonable alternative)
7. Present shortlist to client (or use internally)
```

### Priority 2: Naming Criteria Checklist (15 min)
Create a reusable evaluation framework:

```
Name Evaluation Scorecard:
□ Memorability: Can someone hear it once and remember it? (1–5)
□ Pronunciation: Is it easy to say aloud? (1–5)
□ Spelling: Can someone spell it after hearing it? (1–5)
□ Relevance: Does it connect to the business's core offering? (1–5)
□ Uniqueness: Is it distinct from competitors? (1–5)
□ Domain: Is .com available? If not, is .co/.io acceptable? (Y/N)
□ Trademark: Quick Google search — any obvious conflicts? (Y/N)
□ Social handles: Is the name available on LinkedIn/X/Instagram? (Y/N)

Minimum score for shortlist: 20/30
```

## Risks & Considerations

1. **Domain availability changes:** A domain shown as available when generated may be registered by someone else before the client decides. **Mitigation:** Advise clients to register the domain immediately upon selection
2. **Trademark conflicts:** Atom checks domain availability but not trademark conflicts. A name may have an available .com but still infringe on an existing trademark. **Mitigation:** Recommend a basic trademark search (via WIPO Global Brand Database or local IP office) before finalising
3. **AI-generated sameness:** AI naming tools can produce similar-sounding names (short, abstract, vowel-heavy). **Mitigation:** Review the full list critically — reject names that sound like every other AI startup. Prioritise names with meaning or connection to the business
4. **Cultural sensitivity:** For Botswana-market clients, ensure generated names don't have unintended meanings in Setswana or other local languages. **Mitigation:** Cross-check shortlisted names with local language speakers
5. **Over-reliance on AI:** The best business names often come from human creativity, wordplay, or local context that AI may not capture. **Mitigation:** Use Atom as a starting point, not the final word. Combine AI-generated options with manually brainstormed names

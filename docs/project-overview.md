# Project Overview

## Purpose

The Edmond Moepswa project is a production personal brand website designed to showcase professional services, portfolio projects, and digital products. It serves as a central hub for lead generation, service discovery, and automated client onboarding.

## Executive Summary

- **Primary Focus**: Web Design, Web Applications, Workflow Automation
- **Core Value**: Providing high-end, custom-built digital solutions with a focus on systems thinking and P&L optimization.

## Tech Stack Summary

| Layer | Technology |
| :--- | :--- |
| **Framework** | Next.js 16 (App Router) |
| **CMS** | Payload CMS 3.82 |
| **Database** | PostgreSQL (Neon) |
| **Payments** | Dodo Payments |
| **Automations** | Make.com |
| **Monitoring** | Sentry |

## Key Features

1.  **Portfolio Showcase**: 8 demo projects with detailed case studies.
2.  **Service Catalogue**: Tiered pricing for web and automation services.
3.  **Interactive Pricing Calculator**: Multi-currency tool for instant project estimates.
4.  **Resource Library**: Free and paid digital downloads (Starter Kits, Guides).
5.  **Automated Lead Capture**: Integration with Make.com for instant nurturing.
6.  **Booking Integration**: Seamless discovery call scheduling via Cal.com.

## Repository Structure

The project is structured as a **Monolith** with a clear separation between business planning, source code, and automated workflows.

- `/src`: Main application code.
- `/business-planning`: Strategic definitions and research.
- `/make-workflows`: Automation pipeline logic.
- `/docs`: Technical documentation.
